# Screenshots

Drop PNG captures here with these exact filenames so the main
[README](../../README.md) picks them up:

| Filename              | What to capture                                                       |
|-----------------------|------------------------------------------------------------------------|
| `dashboard.png`       | Full Dashboard tab — summary cards, earnings chart, holdings, donut    |
| `earnings-chart.png`  | Just the **Cumulative Earnings** chart panel (close-up)                |
| `trades.png`          | Trades tab with the filter bar visible and 2-3 rows shown              |
| `data-tab.png`        | Data tab showing Export/Import buttons and the "Last export" card      |

## How to capture cleanly

1. Open <http://127.0.0.1:5173> in a browser at ~1280px wide
2. Use the OS screenshot tool:
   - **Windows:** `Win + Shift + S`, drag a region
   - **macOS:** `Cmd + Shift + 4`, drag a region
3. Save the file with the exact name above into this folder
4. Commit and push — the README on GitHub will render the images
   automatically

## Tips

- For dashboard captures, scroll to the very top first so the hero
  "Total Earned" card is fully visible.
- For chart captures, hover over a date to include the tooltip — it
  shows off the styling.
- Crop tightly. Avoid including the browser chrome unless it's
  intentional.
